
# Sudoku React App

This is a Sudoku game built with React and TypeScript.

## Live Demo

The latest version is automatically deployed to GitHub Pages:

- [https://SuezD.github.io/sudoku](https://SuezD.github.io/sudoku)

---

## Development

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).


## Available Scripts


### `npm start`
Runs the app in development mode at [http://localhost:3000](http://localhost:3000).


### `npm test`
Runs the test suite using Jest and React Testing Library.


### `npm run build`
Builds the app for production to the `build` folder.


### `npm run deploy`
Runs tests, builds the app, and deploys it to GitHub Pages using the `gh-pages` branch.

---

## CI/CD & Deployment

This project uses GitHub Actions to automatically:
- Run all tests
- Build the app
- Deploy to GitHub Pages on every push to the `main` branch

You can find the workflow in `.github/workflows/deploy.yml`.
